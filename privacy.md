Demand Clicker does not collect, store, or share any personal data.

1. Information Collection

This extension does not collect or transmit:

Personal information

Browsing history

Webpage contents

Usage analytics

The only information stored locally is a small toggle flag (enabled: true/false) that remembers whether the extension is turned on. This data never leaves your device.

2. Permissions

Demand Clicker requests the following Chrome permissions strictly to perform its function:

storage: to remember the enabled/disabled toggle

scripting, activeTab, and host permissions: to detect and click matching buttons on webpages you visit

These permissions are used only locally. The extension never sends or shares page data externally.

3. Data Sharing

No data is shared with anyone, and the extension communicates with no external servers.

4. Contact

If you have questions about this privacy policy, you can reach out on github issues.
